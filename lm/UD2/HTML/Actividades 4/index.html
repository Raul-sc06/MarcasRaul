1. Atributos del elemento <area> en HTML5

En HTML5, el elemento <area> sigue siendo utilizado dentro de un elemento <map> para definir regiones interactivas de una imagen. Los atributos que soporta en HTML5 son los siguientes:

Atributos Globales

Todos los atributos globales de HTML5 pueden ser utilizados en <area>, como:

  • id
  • class
  • style
  • title
  • lang, etc.

Atributos específicos de <area> en HTML5

1. alt (obligatorio)

  • Proporciona texto alternativo para el área interactiva.
  • Mejora la accesibilidad describiendo la funcionalidad o el contenido del área.

2. coords

  • Especifica las coordenadas de las áreas definidas.
  • Formato de las coordenadas según el atributo shape:
    • Para un rectángulo (rect): x1, y1, x2, y2.
    • Para un círculo (circle): x, y, r (centro y radio).
    • Para un polígono (poly): x1, y1, x2, y2, ..., xn, yn.

3. href

  • Define la URL o enlace del área.

4. shape

  • Define la forma del área interactiva:
    • default: Cubre toda la imagen (área predeterminada).
    • rect: Define un rectángulo.
    • circle: Define un círculo.
    • poly: Define un polígono.

5. target

  • Especifica dónde abrir el enlace del área:
    • _self: En la misma ventana/pestaña.
    • _blank: En una nueva ventana/pestaña.
    • _parent, _top.

6. rel

  • Describe la relación entre el documento actual y el destino del enlace.
  • Valores posibles: nofollow, noopener, noreferrer, etc.

7. type

  • Define el tipo MIME del recurso vinculado.

8. download

  • Indica que el recurso vinculado debe descargarse en lugar de abrirse.
  • Se puede especificar un nombre de archivo opcional.

9. referrerpolicy

  • Controla cómo se maneja el encabezado HTTP Referer al seguir el enlace.
  • Valores comunes: no-referrer, origin, unsafe-url, etc.
